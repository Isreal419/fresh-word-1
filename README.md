# FRESHWORD MIRACLE CENTER INTERNATIONA Website

A modern, responsive church homepage built with Express, EJS, CSS, and MongoDB.

## Setup

1. Copy `.env.example` to `.env` and update the MongoDB URI if needed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000`

## Features

- Responsive homepage with hero, schedule, sermons, events, pastor message, testimonials, donation CTA, and prayer request form
- Express + EJS template structure
- MongoDB storage for prayer requests and newsletter subscribers
- Sticky navigation, smooth scrolling, and hover animations
